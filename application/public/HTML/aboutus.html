About Team

The team behind this website is composed of San Francisco State University Juniors and Seniors in the Computer Science Department. This project was constructed 100% virtually due to the COVID-19 pandemic. Our team heavily relied on communication and resources such as GitHub and Zoom video-conferencing.

About Website

This project website is designed to be an online marketplace for San Francisco State University students, staff, and alumni only. The purpose behnd this project is to provide an easy and efficient way for members to sell and purchase items in a safe and secure manner with as little physical contact as possible.
